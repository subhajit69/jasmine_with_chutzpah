Introduction

With the using modern javascript framework like Jquery, Knockout, Angular js, Node.js; the client side version of the web application have been complex. Everyday more and more functional and business logic we are putting in client side js to make our application more faster, robust, responsive and overall good user friendly. So, unit testing for javascript functions have became more essential for developer to deliver a quality product. In this context, It is very useful to use Jasmine and Chutzpah Test Adapter for visual studio to javascript independent of framework. This article is all about configuring and using Jasmine and Chutzpah.

Environment

  • Visual Studio : v2013 - v2015
  • Jasmine
  • Chutzpah Test Adapter : 4.2.3
